 CARACTERÍSTICAS CLAVE DEL SISTEMA
════════════════════════════════════════════════════════════════════════════════

✅ Sin restricciones semestrales  → Navegar/reservar todo el año
✅ Formulario simplificado        → Sin campos innecesarios (tel, correo, software)
✅ Responsable flexible           → Cualquier persona, sin catálogo cerrado
✅ Traslape en tiempo real        → Alerta ANTES de completar el formulario
✅ Calendario maestro             → Festivos y cierres integrados
✅ Eliminación masiva             → Un solo comando para múltiples reservas
✅ Historial completo             → Activas, pasadas y canceladas
✅ Recurrencia automática         → Semanal, quincenal, mensual
✅ Asistente IA                   → Reservar con lenguaje natural
✅ Dashboard KPIs                 → Métricas de uso para la patrocinadora
✅ Exportación PDF/Excel          → Para directivos sin acceso al sistema
✅ Respaldo automático            → Datos seguros en la nube
